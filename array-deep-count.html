Array.prototype.length даст вам количество элементов верхнего уровня в массиве. Ваша задача - создать функцию deepCount, которая возвращает количество ВСЕХ элементов в массиве, включая любые внутри массивов внутреннего уровня. Например: deepCount ([1, 2, 3]); // >>>>> 3 deepCount (["x", "y", ["z"]]); // >>>>> 4 deepCount ([1, 2, [3, 4, [5]]]); // >>>>> 7 Вход всегда будет массивом.
