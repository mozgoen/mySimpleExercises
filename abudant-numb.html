Обильное число или избыточное число - это число, для которого сумма его собственных делителей больше самого числа. Целое число 12 является первым обильным числом. Его собственные делители равны 1, 2, 3, 4 и 6 в общей сложности 16 (> 12). Напишите ф-ю abundantNumber(num)которая returns true if num is abundant, false if not.
