Анаграммы – слова, состоящие из одинакового количества одинаковых букв, но в разном порядке. Например: воз - зов киборг - гробик корсет - костер - сектор Напишите функцию aclean(arr), которая возвращает массив слов, очищенный от анаграмм. Например: var arr = ["воз", "киборг", "корсет", "ЗОВ", "гробик", "костер", "сектор"]; alert( aclean(arr) ); // "воз,киборг,корсет" или "ЗОВ,гробик,сектор" Из каждой группы анаграмм должно остаться только одно слово, не важно какое. Чтобы обнаружить анаграммы, разобьём каждое слово на буквы и отсортируем их. В отсортированном по буквам виде все анаграммы одинаковы. Например: воз, зов -> взо киборг, гробик -> бгикор ...
