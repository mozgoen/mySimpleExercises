Вам нужно написать функцию, которая возвращает первый не повторяющийся символ в данной строке. Например, для функции string "test" следует вернуть 'e'. Для функции string «teeter» должна возвращаться «r». Если строка содержит все уникальные символы, верните только первый символ строки. Пример: для функции «тренда» вход должен возвращать 't' Вы можете предположить, что входная строка всегда имеет ненулевую длину.
