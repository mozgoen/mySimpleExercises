Ваша задача - написать функцию, которая возвращает сумму следующих рядов до n-го члена (параметра). Серия: 1 + 1/4 + 1/7 + 1/10 + 1/13 + 1/16 + ... Правила: Вам нужно округлить ответ до 2 десятичных знаков и вернуть его как String. Если заданное значение равно 0, оно должно вернуть 0,00 В качестве аргументов вам будут даны только натуральные числа. Примеры: SeriesSum (1) => 1 = "1.00" SeriesSum (2) => 1 + 1/4 = "1,25" SeriesSum (5) => 1 + 1/4 + 1/7 + 1/10 + 1/13 = "1.57"
