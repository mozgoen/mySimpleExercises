Вам предоставляется массив положительных ненулевых int и int n, представляющий n-ю степень (n> = 2). Для данного массива вычислите сумму каждого значения на n-ю степень. Затем вычтите сумму исходного массива. Пример 1: Вход: {1, 2, 3}, 3 -> (1 ^ 3 + 2 ^ 3 + 3 ^ 3) - (1 + 2 + 3) -> 36 - 6 -> Выход: 30 Пример 2: Вход: {1, 2}, 5 -> (1 ^ 5 + 2 ^ 5) - (1 + 2) -> 33 - 3 -> Выход: 30
