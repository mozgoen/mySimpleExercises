Вы должны реализовать функцию maxDiff, которая возвращает разницу между самым большим и наименьшим значением в списке (lst), полученном как параметр. Список (lst) содержит целые числа, что означает, что он может содержать некоторые отрицательные числа. Если список пуст или содержит один элемент, верните 0. Список (lst) не отсортирован. maxDiff ([1, 2, 3, 4]); // возвращаем 3, потому что 4 - 1 == 3 maxDiff ([1, 2, 3, -4]); // возвращаем 7, потому что 3 - (-4) == 7
