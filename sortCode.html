Сортировка подарочного кода Напишите функцию sortGiftCode / sort_gift_code / SortGiftCode, которая принимает строку, содержащую до 26 уникальных алфавитных символов, и возвращает строку, содержащую те же символы в алфавитном порядке. Примеры: sortGiftCode ('abcdef'); // => возвращает 'abcdef' sortGiftCode ('pqksuvy'); // => возвращает 'kpqsuvy' sortGiftCode ('zyxwvutsrqponmlkjihgfedcba'); // => возвращает 'abcdefghijklmnopqrstuvwxyz'
