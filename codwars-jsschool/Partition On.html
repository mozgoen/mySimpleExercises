Напишите функцию, которая разбивает список элементов на основе заданного предиката. После запуска функции разбиения список должен иметь вид [F, F, F, T, T, T], где Fs (соответственно Ts) - это элементы, для которых функция предиката возвращает false (соответственно true). ПРИМЕЧАНИЕ. Разделение должно быть стабильным; другими словами: упорядочение Fs (соответственно Ts) должно сохраняться относительно друг друга. Для удобства и полезности функция разбиения должна возвращать индекс границы. Другими словами: индекс первого значения T в элементах. Например: var items = [1, 2, 3, 4, 5, 6]; функция isEven (n) {return n% 2 == 0} var i = partitionOn (isEven, items); // элементы должны теперь быть [1, 3, 5, 2, 4, 6] // теперь i должен быть 3
